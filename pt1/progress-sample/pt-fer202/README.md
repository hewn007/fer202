# Student Tuition Payment System

This project is a Student Tuition Payment System built with React. It provides a user-friendly interface for students to manage their tuition payments, including a login form, dashboard, and payment filtering options.

## Features

- User authentication with a login form
- Dashboard UI displaying payment information
- Navigation bar for easy access to different sections
- Filter options for sorting payment data

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/pt-fer202.git
   ```

2. Navigate to the project directory:
   ```bash
   cd pt-fer202
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage

To start the application, run:
```bash
npm start
```

The application will be available at `http://localhost:3000`.
